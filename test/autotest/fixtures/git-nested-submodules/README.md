# Git Nested Submodules Fixture

`test/autotest/create-nested-git-submodule-fixture.mjs` materializes a five-level nested Git submodule tree under `runtime/` for Git History and Git Diff automation.

The generated repositories are ignored by Git and can be recreated at any time by running the fixture script.
